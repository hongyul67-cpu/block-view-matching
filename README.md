# 🧩 블록 뷰 매칭

정투상도 감각을 기르는 연습 게임입니다. 입체 블록과 투상도(정면도·평면도·측면도)를
맞춰 보며 도면 읽기의 기초를 다집니다. 자동화설비·기계 계열 학습의 기본기용입니다.

🔗 **바로가기:** https://hongyul67-cpu.github.io/block-view-matching/

## 주요 기능
- 입체 블록 ↔ 투상도 매칭 연습
- 정면도·평면도·측면도 관계 이해
- (부록) `투상도 연습 문제.html` — 추가 연습 문제

## 결과수집
학습 결과를 교사 구글시트로 제출할 수 있습니다. 공유 링크에 `?rc=`(교사 exec 주소)가
붙으면 제출 버튼이 활성화됩니다. — 배포는 "링크모음 마스터"에서 합니다.

## 사용 방법
- 위 주소를 열면 바로 사용할 수 있습니다. 설치가 필요 없습니다.
